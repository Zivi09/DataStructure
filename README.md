# 🧠 Data Structures Implemented

## 📦 Linear Data Structures

| Data Structure | Description |
|---|---|
| Stack | LIFO structure with push/pop operations |
| Queue | FIFO structure with enqueue/dequeue |
| Doubly Linked List | Nodes with previous and next pointers |
| Priority Queue | Elements served based on priority |

---

## 🌳 Non-Linear Data Structures

| Data Structure | Description |
|---|---|
| Binary Tree | Tree structure with left and right children |
| Graph | Vertex-edge based structure |
| Depth First Search (DFS) | Graph traversal algorithm |

---

## #️⃣ Hashing

| Implementation | Description |
|---|---|
| Basic Hash Table | Simple hashing implementation |
| Hash Table with Chaining | Handles collisions using linked lists |

---

## ⚙️ Algorithms Implemented

| Algorithm | Purpose |
|---|---|
| Traveling Salesman Problem (TSP) | Finds shortest route between cities |
| Huffman Coding | Data compression algorithm |
| Depth First Search | Graph traversal |
