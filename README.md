# 🧠 Data Structures Implemented

### 📦 Linear Data Structures
*   **Stack**: LIFO structure with push/pop operations.
*   **Queue**: FIFO structure with enqueue/dequeue.
*   **Singly/Doubly Linked List**: Dynamic node-based structures.
*   **Priority Queue**: Elements served based on priority.

### 🌳 Non-Linear Data Structures
*   **Binary Tree**: Standard BST with insert, delete, and traversals.
*   **Graph**: Adjacency list representation with vertex/edge management.
*   **BFS/DFS Tree**: Generation of traversal trees from graphs.

### #️⃣ Hashing
*   **Basic Hash Table**: Simple hashing implementation.

### ⚙️ Algorithms
*   **Traveling Salesman Problem (TSP)**: Brute-force solution for finding shortest routes.
*   **Huffman Coding**: Frequency-based data compression.

# 🚀 Getting Started

## 🐍 Python Version
1. Clone the repository:
   ```bash
   git clone https://github.com/Zivi09/DataStructure.git
   cd DataStructure
   ```
2. Run any CLI file:
   ```bash
   python CLI/filename.py
   ```
3. Run the GUI application:
   ```bash
   python Visualizer_Tkinter.py
   ```

## ☕ Java Version
1. Navigate to the Java folder:
   ```bash
   cd Java
   ```
2. Compile and run any file using `java`:
   ```bash
   java FileName.java
   ```
   *Note: All Java implementations are interactive and will prompt for inputs.*
